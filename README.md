# **模型介绍**

竹智是由竹间智能使用中英文对话数据微调的中英双语对话模型。训练时基本上沿袭官方的多轮对话数据组织格式，具有对话流畅、部署门槛较低等众多优秀特性：

- 较长的上下文：在对话阶段使用 8K 的上下文长度训练。
- 更高效的推理：在 INT4 量化下，6G 显存支持的对话长度由 1K 提升到了 8K，推理速度也有较大的提升。
